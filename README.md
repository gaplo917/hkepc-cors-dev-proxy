# Introduction 
HKEPC Development CORS proxy for faster development on HKEPC Ionic Reader

## Setup
```
yarn install

yarn start
```


## Test Proxy

```
http://0.0.0.0:1337/www.google.com

http://0.0.0.0:1337/www.hkepc.com/forum
```

## Modify Port & Host
go to `package.json` > `scripts` > `start`
